# ts-practice-with-bun

To install dependencies:

```bash
bun install
```

To run:

```bash
bun run index.ts
```

This project was created using `bun init` in bun v1.1.8. [Bun](https://bun.sh) is a fast all-in-one JavaScript runtime.

## bun add
bun add -d eslint @eslint/js @types/eslint__js typescript typescript-eslint
